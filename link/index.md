## 友链申请

{% tabs 友链申请, 1 %}

<!-- tab 友链要求 -->

- [ ] 网站内容符合**中国大陆法律法规**
- [ ] **已添加本博客的友情链接** 原则上不接受单向友链
- [ ] 网站类型为**技术类个人博客**
- [x] 为了能尽快添加上友链，评论前请将昵称、邮箱、网站补充完整
- [x] 原则上建站至少三个月，文章数不得少于 10 篇

<!-- endtab -->

<!-- tab 申请友链 -->

参照以下格式留言评论即可

```yaml
- name: #站点名称
  link: #站点链接
  avatar: #站长头像
  descr: #站点描述
  siteshot: #站点截图
```

<!-- endtab -->

<!-- tab 本站信息 -->

- name: s1rius' blog
